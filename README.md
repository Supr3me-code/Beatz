# Beatz
Music app  

<img src="ReadMe/headphone.png">  


  
A cross-platform app that extracts all the .mp3 files in the device and becomes the super audio app.

## Packages used:
audioplayers: ^0.18.3  
path: ^1.8.0  
path_provider: ^2.0.1  
flutter_file_manager: ^0.2.0  
path_provider_ex: ^1.0.1
